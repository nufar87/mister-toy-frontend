## Frontend Tech

- Vue.js
- Vuex
- Axios
- Element-plus,
- Socket.io-client
- SCSS

## Features

- Toys CRUD
- Filter
- Sort
- Search
- Dashboard with Charts
- Map with location of stores
- Login, signup and logout
- Chat rooms by topic
